<b>Step 1</b>: install laravel packages<br>
<b>Step 2</b>: run "npm install"<br>
<b>Step 3</b>: run "npm run dev"<br>
<b>Step 4</b>: run migration command to create tables<br>
<b>Step 5</b>: create an account<br>
